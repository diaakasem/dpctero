Pricing Table Builder

Pricing Table Builder is a simple way to create your own style and content quickly. It save you time.

It's an easy way to customize your admin content.

Go to Pricing Table Builder

How to customize my pricing table?

Add new plan

You can add new plan by clicking on builder button. You can have 5 plans maximum.

Remove existing plan

You can remove a plan by clicking on trash icon below each plan.

Reorder plan

If you want to reorder plan, just click on drag icon at the top of each plan and drag it to position you want.

Modify plan Layout

If you want to reorder plan, just click on drag icon at the top of each plan and drag it to position you want.

Remove item inside plan

You can remove a item in each plan by clicking on trash icon at the right side of each item.

Reorder item inside plan

If you want to reorder item, just click on drag icon at the left side of each item and drag it to position you want.

Edit text and icons

If you click on text elements, you will be able to change text.

If double click, a simple editor will appear with various options: size, font weight, color...

Export pricing table template

When you have finished editing your pricing table, you are ready to export it.

Just click on button Get HTML in builder sidebar.

A modal window will appear with html content.

Move your file

Copy and past html content into one of your page in admin folder.
